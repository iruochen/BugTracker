### 3. 下一步思路

- 点击获取验证码
  - 获取手机号
  - 向后台发送ajax
    - 手机
    - tpl=register
  - 向手机发送验证码（ajax/sms/redis）
  - 验证码失效处理 60s
    1. 用户在前端填入手机号，发送验证码
    2. 手机号通过后端调用腾讯云短信api，给用户发送验证码，同时我们将手机号和验证码存入redis中，设置超时时间为 60s
    3. 用户接收到验码在前端页面中填写
    4. 我们获取用户输入的验证和redis中的存储的进行比较
