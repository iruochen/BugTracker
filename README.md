# BugTracker

## 技术栈
- 前端：HTML/BootStrap/JQuery/JS
- 后端：Django
- 数据库：MySQL/Redis

## 项目基本功能
- 用户注册登录（手机号/用户名）
- 项目管理（新建、展示、星标等）
- 统计概览（日期范围组件 daterangepicker、highcharts）
- Wiki文档管理（添加、预览、修改、删除、引入 editor-md 编辑器、腾讯云 COS 上传）
- 文件管理（文件上传、展示、目录切换、上传至腾讯云 COS）
- 问题管理（标题、内容、状态、优先级、关注者、开始结束时间等）
- 支付订单管理（对接支付宝沙箱开发平台）
